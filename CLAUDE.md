# Claude Code Contributions

This repository includes contributions generated with assistance from Claude Code.

## Commit Guidelines

All commits in this repository should follow these guidelines:

1. **Conventional Commits Format**: Use the [Conventional Commits](https://www.conventionalcommits.org/) specification for all commit messages:
   - `feat:` for new features
   - `fix:` for bug fixes
   - `docs:` for documentation changes
   - `test:` for test additions or modifications
   - `refactor:` for code refactoring
   - `chore:` for maintenance tasks

2. **No Attribution**: Commits should NOT include attribution lines such as:
   - `Co-Authored-By: Claude <noreply@anthropic.com>`
   - "Generated with Claude Code"
   - Any similar attribution or authorship claims

The focus should be on the technical content of the change, not on who or what produced it.

## Example Commit Messages

Good examples:
```
feat: implement conformance test harness for Haskell compatibility
fix: correct refcount handling in SsTableHandle Clone implementation
test: add range query validation tests
```

Bad examples:
```
Add feature 🤖 Generated with Claude Code
Fix bug

Co-Authored-By: Claude <noreply@anthropic.com>
```
